Kebijakan Privasi

PropertiKu — terakhir diperbarui: April 2026

PropertiKu adalah aplikasi manajemen properti sewaan yang berjalan di browser Anda (termasuk saat diinstal sebagai PWA). Kami merancang app agar data operasional utama Anda disimpan di perangkat Anda, bukan di server kami.

Data yang disimpan di perangkat Anda

Data seperti unit, penyewa, pembayaran, pengaturan tampilan, dan preferensi disimpan di localStorage browser Anda. Kami tidak mengirim data ini ke server PropertiKu secara default.

Telegram (opsional)

Jika Anda mengisi Bot Token dan Chat ID di Pengaturan, aplikasi Anda mengirim permintaan langsung ke API Telegram. Token dan data chat tidak disimpan di server kami; mereka hanya ada di perangkat Anda dan di infrastruktur Telegram sesuai kebijakan Telegram.

Service Worker & cache

Aplikasi dapat menggunakan Service Worker untuk cache aset agar lebih cepat dan dapat dipakai offline sebagian. Cache tidak menyimpan “data bisnis” Anda di luar localStorage kecuali salinan aset app (JS/CSS/HTML).

Font & sumber daya pihak ketiga

Font dapat dimuat dari Google Fonts. Saat dimuat, penyedia font dapat melihat alamat IP Anda seperti permintaan web pada umumnya.

Analytics & pemantauan error (opsional)

Pengelola situs dapat mengaktifkan Google Analytics dan/atau Sentry dengan mengisi ID di konfigurasi deploy. Jika diaktifkan, data teknis penggunaan (misalnya peristiwa halaman atau error) dapat diproses oleh penyedia tersebut. Saat tidak diaktifkan, tidak ada pengiriman analytics dari kode default PropertiKu.

Backup

Fitur Export/Import memungkinkan Anda menyalin data ke file. Anda bertanggung jawab atas keamanan file tersebut.

Perubahan

Kami dapat memperbarui kebijakan ini. Tanggal di atas akan diubah saat ada revisi material.

Kontak

Untuk pertanyaan privasi, hubungi kontak dukungan yang Anda cantumkan di situs distribusi PropertiKu.

Privacy Policy

PropertiKu — last updated: April 2026

PropertiKu is a rental property management app that runs in your browser (including when installed as a PWA). Operational data is kept on your device by default.

Data stored on your device

Units, tenants, payments, UI settings, and preferences are stored in localStorage. We do not send this data to PropertiKu servers by default.

Telegram (optional)

If you connect a bot, your browser calls Telegram’s API directly. Credentials stay on your device and are subject to Telegram’s policies.

Service worker & cache

A service worker may cache app assets for speed and partial offline use. Business data remains in localStorage unless you export it.

Third-party fonts

Fonts may load from Google Fonts; the provider may process IP addresses as with normal web requests.

Analytics & crash reporting (optional)

Deployers may enable Google Analytics and/or Sentry via configuration. When disabled, the default app does not send analytics.

Backup

Export/import JSON is under your control; protect exported files.

Changes

We may update this policy; the date above will change for material revisions.

Contact

Use the support contact published with your distribution of PropertiKu.
